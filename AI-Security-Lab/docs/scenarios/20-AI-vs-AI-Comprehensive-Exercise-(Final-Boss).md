## 🛠️ 2. Environment Setup
> **⚠️ Prerequisite — CALDERA must be built first.** It has no official Docker Hub image,
> so clone & build it before starting the `offense`/`all` profile:
> ```bash
> git clone https://github.com/mitre/caldera.git --recursive external/caldera
> ```
> Without this, `docker compose --profile all up` will fail to build the `caldera` service.

1. SSH into the GCP lab VM via IAP and ensure Docker is running (`docker ps`).
2. Execute the following command to prepare the target environment:
```bash
# Build CALDERA source first (see prerequisite above), then:
docker compose -f infrastructure/docker-compose.yml --profile all up -d
# Start CALDERA agent in autonomous mode + start AI SOAR playbook (scripts/soar_playbook.py).
```
3. Ensure Splunk HEC is receiving logs over the VPC-internal network.
